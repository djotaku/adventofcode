## Things I learned during this year's Advent of Code
- The right datatype for part 2 (usually involving recursion) is almost always a dictionary. Lists and dequeues are not performant.
- Unit testing is crucial for making sure each little chunk of the example works. That said:
- the example rarely covers corner cases. There were at least 2-3 days when I had code that worked for the example, but not my input.
- CS concepts repeat throughout the challenge, so it helps to look through the subreddit when you have an issue and see if you can 
figure it out for when it appears again.
  
Haven't read yet, but plan to read in preparation for next year: https://auth0.com/blog/advent-of-code-tips-tricks/
